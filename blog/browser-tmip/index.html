Browser Tmip

We can play a lot of music by allowing certain tags of audio, so this is an attractive solution for all those who need maximum flexibility.

This plugin can be started in WebPlayer by using &quot;From tag&quot; and will play music in all styles, including M4A, MP3, Ogg, AAC, FLAC etc.

You can save and open tag / in loop list in different ways, and you can decide which tag will be played.

It supports all major audio encoders such as FLAC, WAV, AC3, M4A, ASF, ALAC, etc. If you want to support them, just edit the tags you want.

If you have MP3 support, you can play an MP3 or any other file or AVI audio in the plugin.
This plugin supports Winamp, Rythym, Spotify, Pandora, Realplayer, etc.
There are 5 presets, with 8 other styles in the playlist, and it support many more.
Some tags can be converted to other tags, but we only support M4A, MP3, AAC, Ogg, FLAC, AAC, etc.
