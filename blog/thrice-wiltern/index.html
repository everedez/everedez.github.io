A ball game
Description

The goal is to get as many points by throwing six balls (cubes) into the water (unless you wish to throw a cube at each player, it will cost two, if you&#x27;re good).

You will only be able to use those eight balls that have not already been used.

To take the ball:

Hit the ball into the water with the hand (no speciality or special effects).
Run up to the top of the water until you hit the water.
Throw the ball back up to the top of the water.
Points are lost if you miss or the ball lands in the water which is almost impossible to do using this game!

You can see all the balls made by the game in the web link on the left hand side, a ten item web page showing all the levels up in this game.

If you go to the left hand web page of the Trowers you will get a little hint of the game by clicking on the level you want, then you&#x27;ll see the details the game offers.
