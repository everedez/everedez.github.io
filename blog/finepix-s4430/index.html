CPU CMP1

CPU CMP1 is a small type technology. This technology removes the drawbacks of previous technologies and solves the current problems that plague current electronic devices. We did not improve the technology. We did however improve the technology itself a bit. This means a CPU without CMP1 cannot only be used in devices that are quite small. It also means that the chip needs a little bit more memory to execute in a small virtual server that runs on a 1.3Ghz machine.

CPU CMP2

CPU CMP2 is also small. This technology can solve many of the problems that plague current systems. This includes an ability to prevent write and read errors and as well a better design of a CPU.

CPU CMP

CPU CMP is the product of the combined efforts of engineers from every discipline that are involved in developing computers. It attempts to remove all such pitfalls. Even if it turns out the solutions are useful just until it runs out of the chip.
